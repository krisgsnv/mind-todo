# Приложение списка дел
## Установка зависимостей

```sh
 npm install
```

## Запуск мини приложения

```sh
 npm run start
```
## Реализован следующий функционал
1. Поле для ввода новой задачи
2. Списки всех задач, невыполненных и выполненных задач (по отдельности)

## Технический стек
1. React (React Hooks)
2. Redux Toolkit
3. TypeScript
4. Antd
